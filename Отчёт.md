# Отчёт по тестированию модуля `triangle.py`

## 1. Цели и задачи тестирования

**Цель тестирования** — проверка корректности работы функций вычисления площади и периметра треугольника, реализованных в модуле `triangle.py`.

**Задачи тестирования:**
- Проверить корректность вычисления площади треугольника.
- Проверить корректность вычисления периметра треугольника.
- Убедиться в корректной работе функций при нулевых значениях аргументов.
- Проверить работу функций с вещественными числами.

---

## 2. Описание тестируемого продукта

Тестируемый продукт — программный модуль `triangle.py`, предназначенный для выполнения математических вычислений, связанных с треугольниками.

Функциональность модуля:
- `area(a, h)` — вычисление площади треугольника по основанию и высоте.
- `perimeter(a, b, c)` — вычисление периметра треугольника по длинам сторон.

Модуль не содержит пользовательского интерфейса и используется как вспомогательный компонент в программных проектах.

---

## 3. Область тестирования

В рамках тестирования проверяются:
- Функция `area`
- Функция `perimeter`

Тестовые сценарии включают:
- Положительные входные значения
- Нулевые значения
- Вещественные значения

Не рассматриваются:
- Проверка существования треугольника
- Обработка некорректных типов данных

---

## 4. Стратегия тестирования

В процессе тестирования применяется:
- **Тип тестирования:** модульное тестирование (Unit Testing)
- **Подход:** автоматизированное тестирование
- **Инструмент:** библиотека `unittest` языка Python

Каждая функция тестируется независимо с использованием набора тестовых случаев, покрывающих основные сценарии использования.

---

## 5. Критерии приёмки

Тестирование считается успешным при выполнении следующих условий:
- Все тесты выполняются без ошибок.
- Фактические результаты соответствуют ожидаемым.
- Не выявлены дефекты в реализованной логике функций.

---

## 6. Ожидаемые результаты

Ожидаемые результаты тестирования:
- Подтверждение корректности работы функций модуля.
- Успешный статус выполнения всех тестов.
- Отсутствие дефектов, влияющих на корректность вычислений.
